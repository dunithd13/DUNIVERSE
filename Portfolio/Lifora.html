LIFORA is a Java-based Blood type compatibility checking system developed to support life-saving decisions in critical medical scenarios. The project offers:
1. Blood Transfusion Compatibility: Identifies compatible donor blood types and Rh factors for safe transfusions.
2. Organ Transplant Compatibility: Checks compatibility using blood type and HLA values for organ transplantation. (HLA- human leukocyte antigen )
3. RhoGAM Vaccine Evaluation: Assesses the need for RhoGAM vaccination during pregnancy based on parental Rh factors.
This is a result of a team work including Thisen Nambukara, Tharidi Gamage and me.

Key Features:
1.Advanced algorithms for compatibility and decision-making.
2.A user-friendly interface built with NetBeans IDE for intuitive interactions.
3.Focused on critical healthcare needs with real-world applications.

Technologies Used: Java, NetBeans IDE and OOP principles.
